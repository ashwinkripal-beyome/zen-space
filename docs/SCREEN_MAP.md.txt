## Navigation Rules
- Authenticated users go directly to their role dashboard.
- Unauthenticated users go to login.
- Role mismatch should redirect to the correct dashboard.
- Locked screens should explain what is restricted and why.

1. Auth screens.

3. OTP and assessment.
4. Report preview.
5. 30-day plan.
6. Therapist dashboard.
7. Admin dashboard last.
