- /app/therapist/clients/:id/report

## Admin Routes
- /app/admin
- /app/admin/users
- /app/admin/clients
- /app/admin/therapists
- /app/admin/reports
- /app/admin/subscriptions

## Route Protection
- All /app routes require authentication.
- Role-based redirects must be enforced.
- Clients cannot access therapist or admin routes.
- Therapists cannot access admin routes.
- Admin can access all routes.

- Auth layout for login and signup.
- Client app layout.
